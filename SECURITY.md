# Security Policy

This repository contains Markdown instructions for AI agents. It does not ship executable code by default.

## Reporting a Security Issue

If you find a security issue in an instruction, example, install command, or future script, please open a private report through GitHub security advisories if available, or create an issue without publishing sensitive details.

Do not include API keys, customer data, private prompts, or confidential business information in public issues or pull requests.

## Scope

Security-relevant issues include:

- instructions that could expose secrets;
- examples that encourage unsafe handling of private company data;
- install commands that point to the wrong repository or unsafe source;
- future scripts or assets that execute unexpected commands.
